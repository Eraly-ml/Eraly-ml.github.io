This project is a real-time smoke detection system using a custom-trained YOLOv5 model. Upon detecting smoke, it automatically sends alerts — including annotated frames — to a specified Telegram chat.

## Features
- Custom-trained YOLOv5 model for smoke detection
- Real-time video processing
- Automatic Telegram alerts with annotated frames
- Fire prevention application
